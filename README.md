# Emotion Classification Modeli (Duyğu Analizi)

Bu layihə tapşırıq üçün hazırlanmış ingilis dilində mətnlərdən 6 emosiyanı (sadness, joy, love, anger, fear, surprise) təyin edən modeldir.

## İstifadə olunan texnologiyalar
- Python
- scikit-learn (TF-IDF + Logistic Regression)
- Gradio (interfeys üçün)

## Nəticələr
- Test accuracy: ~0.85 
- Dataset: test.xlsx
